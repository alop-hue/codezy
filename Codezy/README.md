# Template Portfolio لتطبيق تعليمي (شبيه بـ Mimo)

## كيفية التعديل على القالب

### 1. تغيير النصوص والمحتوى
- افتح ملف `index.html`
- عدل النصوص في:
  - اسم التطبيق (logo و footer)
  - الوصف في القسم الرئيسي
  - المميزات
  - التقنيات المستخدمة
  - معلومات التواصل

### 2. إضافة لقطات الشاشة
- ضع صور التطبيق في مجلد `assets/screenshots/`
- سمها: `screen1.png`, `screen2.png`, `screen3.png`, `screen4.png`
- غير الأسماء في الـ HTML إذا أردت

### 3. تغيير الألوان
- افتح `css/style.css`
- ابحث عن `#667eea` و `#764ba2` (الألوان الرئيسية)
- استبدلها بألوانك المفضلة

### 4. إضافة روابطك
- غير رابط GitHub في زر "استعرض الكود"
- غير روابط التواصل الاجتماعي في الفوتر
- غير رابط البريد الإلكتروني

### 5. إضافة معلومات مشروع Java الخاص بك
في قسم التقنيات، أضف التفاصيل الفنية:
- نسخة Java المستخدمة
- إطار العمل (Spring Boot، Android SDK، إلخ)
- قاعدة البيانات
- وثائق API إذا وجدت

### 6. نشر القالب
- استخدم GitHub Pages أو Netlify أو Vercel للنشر المجاني
